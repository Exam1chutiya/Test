Section B – Q B4

Brute-force Attack:

  • Online: Slow, detectable, guessing passwords live.
  • Offline: Fast, cracks stolen hashed passwords.

Replay Attack: Reusing captured authentication data.

Weak Implementation Attack: Exploiting outdated algorithms or poor key management.

Section C – Q C1 (a)

Static Analysis: Inspects code without running it; detects signatures and patterns; fast but weak against obfuscation.

Dynamic Analysis: Executes malware in sandbox; reveals runtime behavior; strong but slower and resource-heavy.

Effectiveness: Static is quick but limited; dynamic is accurate but costly; hybrid gives best detection.
